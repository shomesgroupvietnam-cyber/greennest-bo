# BMad Output

This folder stores BMad workflow artifacts for GreenNest BuildFlow.

It is not the canonical documentation root. Canonical product and engineering documentation remains in:

- `blueprint/`
- `docs/`

Use this folder for traceable planning, implementation and review artifacts created while requirements evolve.

## Folders

- `planning-artifacts/` - PRD updates, UX plans, architecture decisions, epics, readiness reports and change proposals.
- `implementation-artifacts/` - approved implementation specs, story notes, checkpoints, investigations and reviews.

See `docs/BMAD_DOCUMENTATION_MAP.md` for the full routing policy.
